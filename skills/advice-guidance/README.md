# Advice & Guidance Skills

These skills are designed for agents that provide recommendations, reviews, and strategic input — without making direct changes to code or files.

Use these prompts when you want an agent to:

- Review code and suggest improvements
- Evaluate architecture and design decisions
- Identify risks, anti-patterns, or areas for refactoring
- Offer best-practice guidance on a topic

## Skills in this category

| File | Purpose |
|------|---------|
| [code-review.md](code-review.md) | Thorough code review with actionable feedback |
| [architecture-guidance.md](architecture-guidance.md) | High-level architectural evaluation and recommendations |
