# Execution Skills

These skills are designed for agents that actively perform tasks — writing code, making changes, running commands, and delivering concrete outputs.

Use these prompts when you want an agent to:

- Implement a feature or fix a bug
- Refactor existing code
- Write or update tests
- Set up configuration or tooling

## Skills in this category

| File | Purpose |
|------|---------|
| [feature-implementation.md](feature-implementation.md) | Implement a described feature end-to-end |
| [test-writing.md](test-writing.md) | Write tests for existing or new code |
